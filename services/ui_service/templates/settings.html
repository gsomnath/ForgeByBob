Setup Instructions

📝 Basic Setup
  1. Get a Gemini AI API key from Google AI Studio
  2. Enter your API key in the form above
  3. Specify a local folder where blog projects will be stored
  4. Optionally set a default publication site
🖼️ Google Cloud Imagen Setup (Optional)
  1. Create a Google Cloud Project and enable the Vertex AI API
  2. Set up authentication:
    • Option A: Create a service account and download the JSON key file
    • Option B: Use gcloud auth application-default login for default credentials
    • Option C: Set the GOOGLE_APPLICATION_CREDENTIALS environment variable
  3. Enter your Project ID and enable Imagen in the configuration above
  4. Optionally specify the path to your service account key file
✅ Final Steps
  1. Click "Save Settings" to apply your configuration
  2. Use "Test Gemini Connection" to verify your Gemini API key works
  3. Use "Test Imagen Connection" to verify your Google Cloud setup (if enabled)
